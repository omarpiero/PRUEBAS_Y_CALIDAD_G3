# Avisos de terceros

Este proyecto utiliza componentes de terceros que no forman parte de la obra propia a registrar ante INDECOPI. Antes de cualquier distribucion publica o comercial, se recomienda conservar una copia de las licencias vigentes de cada dependencia.

## Backend

- Laravel Framework
- Laravel Tinker
- Stripe PHP SDK
- FakerPHP, PHPUnit, Mockery, Collision, Laravel Pint, Laravel Sail y Laravel Pail en entorno de desarrollo

## Frontend

- React
- React DOM
- Vite
- Laravel Vite Plugin
- Tailwind CSS Vite
- Chart.js
- Quill
- SortableJS
- Concurrently en entorno de desarrollo

## Activos externos

- Google Fonts: Poppins
- Google Gemini API como servicio externo configurado mediante `GEMINI_API_KEY`
- Certificados CA de curl.se en `storage/certs/cacert.pem` para validacion TLS local

## Criterio de registro

El expediente de derecho de autor debe describir como obra propia el codigo, la arquitectura, la documentacion funcional/tecnica y los elementos visuales creados para esta plataforma. Las librerias, frameworks, APIs y servicios externos deben identificarse como dependencias, no como autoria propia.
